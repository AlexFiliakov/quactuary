quactuary.future package
========================

The future package contains experimental and next-generation actuarial modeling
capabilities under active development. These modules represent cutting-edge
techniques and may have API changes between versions.

.. warning::
   Modules in this package are experimental and subject to change.
   Use with caution in production environments.
